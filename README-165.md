## Description du projet

Ce projet consiste à migrer une application de gestion de tâches (TodoApp) d’une base de données MySQL vers MongoDB. Le backend, développé en Node.js avec Express. Le projet inclut également la gestion des utilisateurs MongoDB avec des permissions spécifiques, ainsi que la mise en place d’une solution de sauvegarde et de restauration de la base de données. L’ensemble de l’application est conteneurisé avec Docker et versionné sur GitHub.

## Instruction de fonctionnement, technologies utilisées

- Node pour le backend
- Vue framework pour le frontend
- Instance de MongoDB sur docker
- Bibliothèque Mongoose pour MongoDB
- Extension MongoDB for VS Code
- Github

## Instruction permettant de faire fonctionner le projet sur un environnement local

- Docker
- Mongo
- Node.js 20+
